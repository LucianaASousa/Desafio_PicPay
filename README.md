# Desafio_PicPay

## Pré-requisitos:
- Ter um IDE instalado (usei o VSCode)
- Instalar o Cypress
> npm install -D cypress
- Abrir o terminal e dentro da pasta desejada clonar o projeto
> git clone https://github.com/LucianaASousa/Desafio_PicPay.git
  
## Para execução:
> npm run cypress:open
